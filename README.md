# founder-ops-toolkit

Claude Code skills for service-based founders to diagnose bottlenecks, extract processes, and build systems that run without them.

## What This Is

Operational skills for the **Extract → Assign → Scale** framework. These skills help you identify what's eating your time, document it, and systematize it so you can step back.

## Requirements

- Claude Code CLI installed
- Pro/Max subscription or API access

## Installation

```bash
npx skills add emillycunha/founder-ops-toolkit
```

Or manually:
1. Clone this repo
2. Add `/skills/operations/` to your Claude Code skills path

## Usage

Describe your operational problem and Claude will route you to the right skill.

**Example triggers:**
- "I'm stuck doing everything" → `bottleneck-finder`
- "How do I document this process" → `sop-generator`
- "What's eating my time" → `time-audit`
- "I need to delegate" → `authority-ladder`
- "Build a system for..." → `process-mapper`
