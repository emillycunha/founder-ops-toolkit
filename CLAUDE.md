# CLAUDE.md

These are operations skills for service-based founders following the Extract → Assign → Scale framework.

See [AGENTS.md](./AGENTS.md) for full contribution guidelines.

## How to Use

When a user describes an operational problem, check `/skills/operations/` for relevant skills.

Each skill lives in its own folder: `skills/operations/skill-name/SKILL.md`

## Before Running

Always announce which skill you're running before executing.

## Skill Discovery

Match user intent to skill descriptions in the frontmatter. Trigger phrases include:
- "I'm stuck doing everything"
- "How do I document this process"
- "What's eating my time"
- "I need to delegate"
- "Build a system for"
